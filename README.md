Recipe Finder Web App

A simple web application that helps users search for recipes and view ingredients and preparation steps. The project focuses on building a clean user interface and practicing modern frontend development using React and JavaScript.

Project Overview

This project allows users to search for recipes based on a keyword. The application fetches recipe data from an external API and displays results in an easy to read format. Each recipe includes the name, image, ingredients, and cooking instructions.

The goal of this project was to practice working with APIs, handling user input, and building reusable components using React.

Features

Search for recipes by keyword

Display recipe images and titles

View ingredients and preparation steps

Responsive layout for desktop and mobile devices

Dynamic data loading from an API

Component based structure using React

Technologies Used

HTML5

CSS3

JavaScript (ES6)

React

Fetch API

How It Works

The user enters a recipe name in the search bar.

JavaScript captures the input and sends a request to a recipe API.

The API returns a list of matching recipes.

React components render the recipe cards dynamically.

The user can view details such as ingredients and instructions.
